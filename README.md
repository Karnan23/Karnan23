<details open>
<summary><b>🧩 TaskFlow AI — AI Productivity & Scheduling Platform</b></summary>
<br/>

AI-powered productivity platform that replaces manual, time-blind task planning with a system that learns each user's real work patterns and auto-generates time-blocked daily schedules.

| Category | Detail |
|---|---|
| **Stack** | React 18 (Vite), Flask, SQLite, Groq API (Llama 3.3 70B) |
| **Scale** | 700-line backend, 9 REST endpoints, 6 tracked task categories |

| **Automation** | Fully automated weekly reports every Sunday via daemon thread |
| **Impact** | Personalized learning engine adjusts AI scheduling using real actual-vs-estimated time ratios |
| **Repository** | [github.com/Karnan23/task-manager](https://github.com/Karnan23/task-manager) |
| **Live Demo** | [task-manager-frontend-sigma-two.vercel.app](https://task-manager-frontend-sigma-two.vercel.app) |

Engineered end-to-end: a Flask backend with a custom `get_category_averages()` learning engine injects each user's actual performance data into every AI prompt, producing schedule recommendations tuned to how the user actually works — not generic estimates.

</details>
